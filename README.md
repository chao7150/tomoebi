# tomoebi
Python3.6+tweepy3.5+PyQt5.6  
ツイッタークライアント。いずれマストドンにも対応する。

# 開発
1.anaconda入れましょう  
2.cloneします  
`git clone https://github.com/chao7150/tomoebi.git`  
`cd tomoebi`  
3.必要なモジュールをインストールします  
`conda env create --file dependencies.yaml`  
4.起動  
`python tomoebi.py`  

モジュールはlinuxじゃないと見つからないかも。
